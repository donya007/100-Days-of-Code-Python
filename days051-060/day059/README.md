## How to Run the Webapp:

1. Navigate to the days051-060/day059/Harrisons-Blog directory in your terminal.

2. Install the package requirements:
    ```
    pip install -r requirements.txt
    ```

3. Run the webapp:
    ```
    python3 server.py
    ```
